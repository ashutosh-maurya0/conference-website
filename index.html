    (IACIDS 2023)

    View More

    IDSCS'23 is a technical gathering of the Data Science, Computer Science and Data security professionals to exchange the latest developments that are gaining momentum in the ever-growing global infrastructure of Computer Science. The conference is a multi-track program scheduled for two days. The conference will feature invited talks and selected paper presentations. IDSCS'23 aims to serve as a technical forum to share and learn about the various works that may lead to the Next generation IT and AI development. IDSCS'23 aims to serve as a technical forum to share and learn about the various works that may lead to the Next generation of Internet Security Infrastructure.

    We invite original research submissions and findings based on large scale practical implementation on the following topics, but not limited to:

    • Data Science
    • Artificial Intelligence
    • Machine Learning
    • Quantum Computing
    • Data & Network Security
    • Computation Security
    • Cyber Security
    • Information Security
    • Secure Multi-Party Computation
    • Tools and Methodologies for design thinking
    • Image Processing
    • Data Analytics
    • Mathematics for Data Science
    • Graph Theory & Application in Data Science
